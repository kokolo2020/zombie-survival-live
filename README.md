# Zombie Survival Live

A YouTube livestream chat game MVP.

## How to deploy on Netlify

1. Upload this folder to GitHub.
2. In Netlify, choose Add new site → Import from GitHub.
3. Select this repo.
4. Deploy.

No build command needed.

## Test commands

Type in the test box:

Mike: !join
Mike: !fight
Sarah: !join
Sarah: !heal
John: !search
John: !build
